# Variáveis de ambiente no Python

## Arquivo ```.env```
Esse arquivo contém as variáveis secretas

## Como importar as variáveis?
```
from dotenv import load_dotenv
import os

load_dotenv()

var = os.getenv('VARIABLE')
```

## Referência
